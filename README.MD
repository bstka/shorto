## How to Run/Install?

```bash
Install dependency!!!
> npm install

Run it!!!
> npm run dev

Test it!!!
> npm run test
```

## How it Works?

### Make Short Url!

```json

    Method: 'POST',
    path: '/'

    payload: {
        "ShortUrl": "YourShortUrl",
        "LongUrl": "YourLongUrl!"
    }

```

If you leave ShortUrl blank, it will generate 5 random char for you!

### Get Redirected

```json

    Method: 'GET',
    path: '/'

    payload: {
        "ShortUrl": "YourShortUrl",
    }

```
